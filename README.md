## Instrucciones para correr el fronted
### 1. Instalar las dependencias

npm install

### 2. Correr el servidor

npm run dev

### 3. Como probar la aplicacion

Para probar el chat correctamente:

- Abre la aplicación en dos navegadores diferentes (por ejemplo, Google Chrome y Firefox)

- Usa dos cuentas de correo distintas en cada navegador, ya que la aplicación está corriendo en local

Nota: El archivo .env utilizado no es el mismo que el del proyecto
